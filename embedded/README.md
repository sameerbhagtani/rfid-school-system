# ESP 12E

- Buzzer

| Wire  | ESP Pin |
| ----- | ------- |
| Black | GND     |
| Red   | D8      |

- LCD

| LCD Pin | ESP Pin |
| ------- | ------- |
| GND     | GND     |
| VCC     | VIN     |
| SDA     | D2      |
| SCL     | D1      |

- RFID Reader

| Reader Pin | ESP Pin |
| ---------- | ------- |
| GND        | GND     |
| 3.3V       | 3V      |
| SCK        | D5      |
| MOSI       | D7      |
| MISO       | D6      |
| RST        | D3      |
| SDA        | D4      |

- Servo Motor (Only in Access Control)

| Wire   | ESP Pin |
| ------ | ------- |
| Brown  | GND     |
| Red    | VIN     |
| Orange | D0      |
